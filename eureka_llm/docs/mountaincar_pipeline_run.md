# MountainCarContinuous-v0: run exactly like Bipedal/Lunar pipeline

This environment is now intended to run through the same multi-round pipeline
(flow: round0 generation -> train -> perception -> analyst -> generator -> reflection),
not via separate shell bootstrap scripts.

## 1) Use provided config
- `configs/ppo_1M_mountaincar_rlzoo.yaml`

Key settings are RL-Zoo-style PPO hyperparameters for MountainCarContinuous-v0.

## 2) Start a new experiment (same pattern as other envs)

```bash
python eureka_llm/framework/pipeline.py new \
  --env-dir eureka_llm/envs/MountainCarContinuous-v0 \
  --exploration eureka_llm/explorations/MountainCarContinuous-v0.json \
  --config eureka_llm/configs/ppo_1M_mountaincar_rlzoo.yaml \
  --rounds 5
```

## 3) Resume experiment

```bash
python eureka_llm/framework/pipeline.py resume \
  --exp-dir eureka_llm/runs/mountaincarcontinuous-v0_<timestamp>_1000000
```

## Notes
- No official reward is needed by this framework loop.
- Round0 reward is generated by the same prompt/template pipeline as other environments.
- Training/evaluation artifact structure is the same as Bipedal/Lunar runs.
